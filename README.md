#  SIEM Implementation and Log Analysis (Splunk)

##  Project Overview
This project demonstrates the implementation of a SIEM solution using Splunk to monitor, analyze, and detect security threats from log data.

##  Objectives
- Analyze 50,000+ log events
- Detect suspicious activities (failed logins, abnormal traffic)
- Create alert rules using SPL
- Simulate SOC (Security Operations Center) workflows

##  Tools Used
- Splunk SIEM
- CSV Log Data
- SPL (Search Processing Language)

##  Project Structure
- logs/ → Contains sample log files
- splunk_queries/ → SPL queries for detection
- alerts/ → Alert rules
- screenshots/ → Dashboard and alerts screenshots
- report/ → Detailed project report
